# Piano on Web 🎹

A simple and interactive web-based piano application.

## 🚀 Features
- Play piano notes using your keyboard or mouse.
- Realistic sound synthesis using **Tone.js**.
- Responsive and clean UI.

## 🛠️ Tools & Technologies
This project was built using the following tools:
- **React 19**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool and development server.
- **Tone.js**: A framework for creating interactive music in the browser.
- **Vanilla CSS**: For styling and layout.
- **Gemini CLI**: This project was developed and managed with the help of **Gemini CLI**, an AI-powered interactive agent.

## 📦 Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
